SkillSwap (PHP + MySQL)
=======================
1) Copy the `skillswap_final` folder to C:/xampp/htdocs/skillswap/
2) Start Apache + MySQL in XAMPP
3) Open phpMyAdmin -> run `bootstrap.sql` to create tables + sample data
4) Visit http://localhost/skillswap/
   - Admin login: admin@skillswap.local / admin123
